# BookMe - Barbershop Management System

BookMe is a modern Android application designed to streamline the appointment booking process for both clients and barbers. The app offers a seamless UI for booking sessions and a robust management dashboard for barbers to handle their daily schedules.

## Developers
- Maya (207371949)

## Key Features
- Smart Booking System: Real-time slot availability based on barber working hours and existing appointments.
- Barber Dashboard: A personalized Today's Appointments view for barbers to manage their daily log.
- Security and Validation: Includes payment simulation with card validation and confirmation dialogs.
- Cloud Integration: Powered by Firebase Firestore for real-time data synchronization.

## How to Use
1. Client Side: Register/Login, select a service, choose a barber, pick a date/time, and complete the simulated payment.
2. Barber Side: Log in with the provided credentials to view today's schedule, block time slots, or cancel appointments.
